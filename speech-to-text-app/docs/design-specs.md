# UI 设计规范

## 配色方案

| 用途 | 颜色值 | Tailwind 类 |
|------|--------|-------------|
| 页面背景 | `#F5F5F5` | `bg-app-bg` |
| 侧边栏背景 | `#FAFAFA` | `bg-app-sidebar` |
| 卡片/内容区背景 | `#FFFFFF` | `bg-white` |
| 边框线 | `#E8E8E8` | `border-app-border` |
| 主文字 | `#333333` | `text-app-text` |
| 次要文字 | `#888888` | `text-app-text-secondary` |
| 强调色（按钮等） | `#4A90D9` | `bg-app-accent` |
| 置顶高亮 | `#FFF8E1` | `bg-app-pin` |
| 删除按钮 | `#E85D5D` | `text-red-500` |
| 成功/已完成 | `#52C41A` | `text-green-500` |

## 字体规范

| 用途 | 大小 | 字重 |
|------|------|------|
| 应用标题 | 14px | 500 |
| 文件列表项标题 | 13px | 400 |
| 文件列表项描述 | 11px | 400 |
| 按钮文字 | 13px | 400 |
| 正文内容 | 14px | 400 |
| 状态栏 | 12px | 400 |

字体族：`"Microsoft YaHei", "PingFang SC", sans-serif`

## 组件尺寸与间距

| 组件 | 规格 |
|------|------|

## 布局结构

```
┌──────────────────────────────────────────────┐
│ 标题栏 (48px)         录音转文字助手           │
├─────────────────┬────────────────────────────┤
│ 侧边栏 (280px)   │  内容区                     │
│                 │                            │
│ [+ 上传] 按钮    │  （选中文件后显示文字内容）    │
│                 │                            │
│ ┌─────────────┐ │                            │
│ │ 📌 文件1    │ │                            │
│ │ 03:25  已完成│ │                            │
│ └─────────────┘ │                            │
│ ┌─────────────┐ │                            │
│ │ 文件2       │ │                            │
│ │ 01:30  未识别│ │                            │
│ └─────────────┘ │                            │
│                 │                            │
│                 │  [导出 TXT] [导出 Word]     │
├─────────────────┴────────────────────────────┤
│ 状态栏 (32px)    已识别 1/2 个文件              │
└──────────────────────────────────────────────┘
```

## 交互规范

- 文件列表项 hover 时背景变为 `#F0F0F0`
- 选中文件列表项边框高亮为 `#4A90D9`（1px 边框）
- 按钮 hover 时透明度降至 0.9
- 删除操作弹出居中模态框确认
- 识别中显示旋转动画（CSS animation）
- 拖拽上传时窗口显示虚线边框提示

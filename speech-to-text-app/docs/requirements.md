# 开发需求文档

## 项目概述
录音自动转文字助手 — 一款 Windows 桌面应用，支持上传音频文件并自动识别转写成文字。

## 用户场景
1. 用户打开软件，看到文件列表（按时间倒序排列）
2. 用户上传音频文件（mp3/wav/m4a 等格式）
3. 用户点击「开始识别」，软件调用在线 API 将音频转为文字
4. 用户在右侧查看识别结果
5. 用户可以导出文字为 TXT 或 Word 文档
6. 用户可以删除不需要的文件，可以置顶重要文件

## 功能需求清单

### F1 — 文件上传
- 点击「上传」按钮，打开系统文件选择器，筛选音频格式
- 支持拖拽音频文件到窗口上传
- 支持格式：mp3, wav, m4a, flac, ogg, wma, aac
- 文件大小限制：单文件 500MB

### F2 — 文件列表
- 按上传时间倒序排列（最新的在上方）
- 置顶的文件始终显示在列表最顶部，带高亮标识
- 每个文件项显示：文件名、时长、上传日期、识别状态标签
- 状态标签：未识别 / 识别中 / 已完成

### F3 — 文件操作
- **删除**：右键菜单或按钮触发，弹出确认对话框，确认后删除音频文件和元数据
- **置顶/取消置顶**：右键菜单或按钮触发，切换置顶状态

### F4 — 语音识别
- 调用 OpenAI Whisper API 进行语音转文字
- 支持中英文混合音频
- 显示识别进度：等待中 → 上传中 → 识别中 → 完成
- 识别结果本地缓存，关闭后重新打开仍可查看
- 识别失败时显示错误信息

### F5 — 文字查看
- 点击文件项，右侧区域显示对应的识别文字
- 支持选中和复制文字内容

### F6 — 导出
- 导出为 TXT：纯文本格式，UTF-8 编码
- 导出为 Word：.docx 格式，保留段落结构
- 导出文件名：`{原音频文件名}_转写结果.txt` 或 `.docx`

### F7 — API Key 管理
- 首次使用时引导用户输入 OpenAI API Key
- Key 加密存储在本地配置文件
- 可在设置中修改 Key
- Key 无效时有明确提示

## 验收标准
1. 应用可在 Windows 10/11 上正常安装和运行
2. 上传、识别、查看、导出全流程无误
3. 关闭重开后数据不丢失
4. UI 布局合理，灰白配色一致
